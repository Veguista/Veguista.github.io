Lady Umbrella

ROLES : Gameplay & Tools Programmer

umbrella > GUN

Action-adventure shooter where players control the special agent Francesca De Angelis. Fight against the mafia and the agency using your multi functional umbrella. Shoot, stun, and hook enemies, and open it to divert bullets.

Studio : Zulo Interactive
Genre : 3rd Person Adventure Shooter
Platform : PC

Steam Page 

Contributions

Cover systems
I developed, documented, and maintained the game's Cover Systems:
  • Coded the movement & camera logic for all characters while taking cover.
  • Built tools for designers to create and place covers in combat encounters.
  • Integrated the animation logic used by characters while in cover.
  • Automated the placement of cover points for enemy IAs.
  • Iterated the system based on the designers' feedback.
other Gameplay elements
I also developed multiple other Gameplay elements:
  • Implemented the shield and animation logic and its health shader.
  • Created a State Machine system to control the player's allowed actions.
  • Built an event-based slow-motion system for gameplay moments.
  • Developed a component-based Actor Pooling system (Not in final build).
Optimization Strategies
I designed and implemented strategies to improve the game's performance:
  • Used profiling to identify performance issues and possible solutions.
  • Implemented fixes in the areas of Level Design, Lighting, and Code.
  • Created guides for other team members to also be able to perform optimizations.
VFX production
I participated in the full VFX pipeline: Creation, optimization, and integration:
  • Used Unreal Engine's Niagara System for particle system generation.
  • Integrated VFXs into Gameplay and configured them for GPU execution.
  • Created VFXs following the art style guide.
